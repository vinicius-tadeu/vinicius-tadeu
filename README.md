## Sobre mim

Sou um desenvolvedor full stack com perfil altamente orientado a produto.

Não fico restrito só ao código da interface ou só ao backend. Trabalho pensando no fluxo completo: arquitetura, experiência do usuário, integrações, automações, performance, monetização e entrega real em produção.

Tenho facilidade para transformar requisitos de negócio em produtos funcionais e escaláveis, principalmente em contextos de:

- SaaS
- dashboards e sistemas internos
- automações com APIs
- integrações com serviços de pagamento
- fluxos com IA
- experiências web modernas com foco em usabilidade
